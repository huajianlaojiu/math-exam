# ➗ 小学数学出卷系统

> **自动生成小学数学试卷 · Automatic Math Exam Generator**

[📦 GitHub](https://github.com/huajianlaojiu/math-exam)

---

## 📖 简介

**小学数学出卷系统** 可根据年级和知识点自动生成小学数学试卷。

## ✨ 功能特性

- 📐 **多题型**：口算、填空、选择、应用题
- 🎯 **按年级**：适配小学 1-6 年级
- 🔢 **难度调节**：可调节题目难度
- 🖨️ **打印友好**：生成的试卷可直接打印
- ✅ **配套答案**：自动生成参考答案

## 🚀 快速开始

\\ash
git clone https://github.com/huajianlaojiu/math-exam.git
cd math-exam
# 打开 index.html 即可生成试卷
